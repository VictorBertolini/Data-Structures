# Estruturas de Dados e Códigos em C

Repositório criado para demonstrar diversas estruturas de dados e códigos em programação dinâmica, recursão e métodos de ordenação feitas em linguagem C, para demonstrar domínio na construção dessas estruturas em uma linguagem mais baixo nível.

--- 

## Estruturas Construídas:

* Lista Encadeada
* Grafos
* Algoritmos de Ordenação
  - Bubblesort
  - Heapsort
  - Insertionsort
  - Selectionsort
  - Shellsort
* Algoritmos Recursivos
